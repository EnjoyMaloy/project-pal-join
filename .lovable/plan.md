

## Объединенный блок валют — план

### Идея

Объединить монеты (A) и токены (S) в один компактный блок с нейтральным фоном, чтобы они не конкурировали с кнопкой "Купить подписку". Блок остается кликабельным и привлекательным, но визуально приглушен.

### Дизайн

```text
┌─────────────────────────────────┐      ┌──────────────────────┐
│  (A) 212,384  │  (S) 2,126,771 │      │ 👑 Купить подписку   │
└─────────────────────────────────┘      └──────────────────────┘
  muted bg, border, subtle icons           яркий violet градиент
```

- **Фон блока**: `bg-muted` (нейтральный серый) с `border border-border` — как у кнопок языка/темы
- **Иконки A и S**: маленькие цветные кружки (фиолетовый для A, оранжевый для S) — сохраняют узнаваемость, но не кричат
- **Текст значений**: `text-foreground`, обычный вес — читаемо, но не акцентно
- **Разделитель**: тонкая вертикальная линия `border-r border-border` между двумя счетчиками внутри блока
- **Высота**: 36px — как у кнопки подписки
- **Hover**: легкое осветление фона, курсор pointer — приглашает кликнуть
- **Кнопка подписки**: остается как есть с ярким violet градиентом — единственный яркий акцент в группе

### Технические изменения

**Файл**: `src/components/Navbar.tsx`

Заменить два отдельных градиентных блока (coins counter + dollar counter) на один объединенный `div` с нейтральным стилем:
- Один контейнер `rounded-full bg-muted border border-border` с двумя секциями внутри
- Между секциями — `div` с `w-px h-4 bg-border` как разделитель
- Иконки: маленькие кружки 20x20 с `bg-[#A66CFF]` (монеты) и `bg-[#E07450]` (токены), opacity ~0.8
- Текст: `text-foreground text-[14px] font-normal`
