# Страница «Ресурсы бренда»

## Что меняем

1. В футере (`src/components/Footer.tsx`) в колонке Company заменить «Карьера / Careers» на «Ресурсы бренда / Brand resources» со ссылкой на `/brand`.
2. Сохранить второй референс (эмодзи-иллюстрации) в `src/assets/brand-emoji-preview.png` через `code--copy`.
3. Создать страницу `src/pages/BrandResources.tsx`.
4. Зарегистрировать роут `/brand` в `src/App.tsx`.

## Структура страницы `/brand`

Светлая страница (`bg-background`), контейнер `max-w-5xl mx-auto px-6 lg:px-10 py-16` с большими отступами между блоками, в стиле существующего дизайна (TT Commons, токены из index.css, мягкие карточки `bg-card`/`border-border`, радиусы `rounded-2xl`).

### Hero (приветствие)
- Маленький caps-eyebrow: «РЕСУРСЫ БРЕНДА»
- H1: «Айдентика нашего бренда»
- Подзаголовок (`text-muted-foreground`, max-w-2xl): «Привет! Здесь собраны материалы, которые помогут вам красиво и единообразно рассказывать о нас. Скачивайте логотип, цветовую палитру и эмодзи-иллюстрации — всё в хорошем качестве и в фирменных цветах.»

### Блок 1 — Логотип
- Двухколоночный layout (`grid lg:grid-cols-[1fr_1.2fr] gap-10`):
  - Слева: H2 «Логотип» + текст «Вы можете использовать логотип для партнёрских материалов. Скачивайте в хорошем качестве и фирменных цветах.» + кнопка `Button` с иконкой `Download` — «Скачать .ZIP».
  - Справа: сетка `grid grid-cols-2 gap-4` с 4 пустыми превью-плейсхолдерами (`aspect-square rounded-xl bg-muted border border-dashed border-border` + мелкая подпись `text-caption-12 text-muted-foreground` по центру: «Скоро»). Эти плейсхолдеры — заглушки под будущие превью лого.

### Блок 2 — Цветовая палитра
- H2 «Цветовая палитра» + рыба «Наша палитра построена на контрасте насыщенного фиолетового, тёплого оранжевого и спокойных нейтралей. Используйте эти цвета как основу любых визуальных материалов о бренде.»
- UI-палитра из референса: 4 swatches в `grid sm:grid-cols-2 lg:grid-cols-4 gap-4`. Каждый swatch — карточка `rounded-2xl overflow-hidden border border-border`:
  - Верх: цветной блок `h-40` с фоном цвета.
  - Низ: белая подложка с padding, название, HEX, RGB, CMYK (моноширинно, `text-caption-12`).
  - Цвета:
    1. **OA Purple** — `#A66CFF` · RGB 166, 108, 255 · CMYK 35, 58, 0, 0
    2. **Active Orange** — `#FF8645` · RGB 255, 134, 69 · CMYK 0, 47, 73, 0
    3. **Warm Neutral** — `#FFFCF5` · RGB 255, 252, 245 · CMYK 0, 1, 4, 0 (с тонкой рамкой т.к. почти белый)
    4. **Graphite Black** — `#202020` · RGB 32, 32, 32 · CMYK 0, 0, 0, 87
- Кнопка `Download` — «Скачать палитру».

### Блок 3 — Эмодзи-иллюстрации
- Двухколоночный layout (`grid lg:grid-cols-[1.1fr_1fr] gap-10 items-center`):
  - Слева: H2 «Эмодзи-иллюстрации» + рыба «Набор объёмных 3D-эмодзи для соцсетей, презентаций и сообщений в Telegram. Используйте их, чтобы добавить тёплый характер брендовым материалам.» + строка из двух кнопок: основная `Download` «Скачать», вторая outline с иконкой `Send` «Добавить в Telegram».
  - Справа: превью-картинка (импорт из `@/assets/brand-emoji-preview.png`) в карточке `rounded-2xl bg-card border border-border p-6` с `object-contain`.

## Поведение кнопок
Кнопки скачивания пока без реальных файлов — `onClick` показывает `toast` («Файл скоро будет доступен») через `useToast`. Кнопка «Добавить в Telegram» — `<a href="https://t.me/" target="_blank">` плейсхолдер. Так пользователь может позже подменить URL/файлы.

## Технические детали
- Использовать только семантические токены и существующие классы typography (`text-h1`, `text-h2`, `text-body-14`, `text-caption-12-caps` и т.п.).
- Иконки из `lucide-react`: `Download`, `Send`, `ArrowLeft`.
- Сверху страницы — ненавязчивая ссылка «← На главную» (`Link to="/"`).
- Адаптив: на мобиле всё в одну колонку, swatches 1→2→4.
- Никаких изменений бэкенда/БД.
